# Sotech Arduino Projects

Sotechコースで作成したArduinoプロジェクトと学習資料です。

## 学習ログ

1か月ごとの学習ログを`study_logs`ディレクトリに保存しています。
[STUDY_LOG](STUDY_LOG.md)にて各月の学習内容を確認できます。

## Projects

プロジェクトはカテゴリごとに`projects`ディレクトリに整理されています：

- **gpio_output**: LED出力などの基本操作
- **gpio_input**: スイッチ入力などの基本操作
- **analog_input**: 可変抵抗などを使用したアナログ入力
- **actuators**: サーボモーターやDCモーターの制御
- **displays**: LCDなどの表示デバイス

## 環境

- Arduino IDE
- VS Code (optional)
